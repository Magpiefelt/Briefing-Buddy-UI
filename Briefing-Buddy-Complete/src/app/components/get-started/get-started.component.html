Briefing Buddy

A short overview of the service. This is a couple sentences that helps the user understand what the service is.

Use this service to apply for briefing assistance. You can use this service to:

  • See if you or a family member is eligible for briefing assistance
  • Create and submit an application for briefing assistance
  • Continue an application for briefing assistance that you already started

Before you begin

The application form should take about 20 minutes to complete.

In order to complete the application you will need:

  • government issued ID for the person applying
