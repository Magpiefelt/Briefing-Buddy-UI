Briefing Buddy

A short overview of the service. This is a couple sentences that helps the user understand what the service is.

Use this service to apply for Briefing Buddy. You can use this service to:

  • See if you or a family member is eligible for Briefing Buddy
  • Create and submit an application for Briefing Buddy
  • Continue an application for Briefing Buddy that you already started

Before you begin

The application form should take about 20 minutes to complete.

In order to complete the application you will need:

  • Government issued ID for the person applying
  • Your Alberta Government email address
